# Deep-Dive-SIRD
This project investigates the behaviors of SIRD congestion control (NSDI 2025), which was evaluated in CloudLab. The ultimate goal is to optimize congestion control for data center and HPC networks.  

SIRD paper: https://www.usenix.org/system/files/nsdi25-prasopoulos.pdf \
SIRD implementation code: https://github.com/epfl-dcsl/SIRD-Caladan-Impl
